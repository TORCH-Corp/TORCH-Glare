  The PassCheck component provides visual feedback on the strength of a password based on specific criteria. It evaluates the password based on length, the presence of symbols, and uppercase letters, and displays different levels of strength: weak, good, and great.

### Usage

```jsx
import { PassCheck } from 'torch-glare';

const App = () => (
  <div>
    <PassCheck value="short" />
    <PassCheck value="good@password" />
    <PassCheck value="Great1@Password" />
  </div>
);

export default App;
```
