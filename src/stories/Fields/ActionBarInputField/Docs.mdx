# ActionBarInputField

`ActionBarInputField` is an input component that can include action buttons, badges, and a dropdown menu.

## Examples

### Default Style

    <ActionBarInputField 
     style={{width:"300px"}}
        name="default"
        placeholder="Enter text here..."
        action_button_children={ <Button component_size="L" left_icon={<i className="ri-mail-download-line"></i>}/>}
    />

## Usage

```jsx
import { ActionBarInputField,Button } from 'torch-glare';

const App = () => (
  <div>
    <ActionBarInputField 
        name="action-buttons"
        placeholder="Enter text here..."
        action_button_children={[
            <Button component_size="L" onClick={() => alert('Action 1')} left_icon={<i className="ri-mail-download-line"></i>}></Button>,
        ]}
    />
  </div>
);

export default App;
```
