# Framework Color System

The Framework Color System leverages the 'glare' core color system along with a flexible theme mapping strategy. This approach ensures that all components within the framework can dynamically adapt to different themes while maintaining consistency in color usage.

## Directory Structure

The colors and themes are organized within the `src/lib/styles/colors` directory:

- `coreColorSystem/`: Houses the `index.css` file with the core color definitions.
- `themes/`: Contains the `torch` and `css` themes files.
- `mods/`: Contains the `dark`, `light`, and `default` mod CSS files.
- `index.css`: The entry point file.

## Core Color System File

The `Glare` core color system consists of a set of predefined pure color variables. These are defined globally and serve as the foundational palette for the entire framework.

**Example of Core Colors:**

```css
:root {
  --purple-100: #F5EEFFFF;
  --purple-200: #EBDCFFFF;
  --purple-300: #D8BAFFFF;
  --medium-red-800: #88071DFF;
  --medium-red-900: #5B0513FF;
  --medium-red-1000: #2D020AFF;
  /* Additional core colors */
}
```

## Themes File

This file contains the TORCH theme file and CSS theme file. These files are used within the mods file to be referenced by the mod files. Note: CSS is a theme name.

### Example of Themes File:

```css
@import './coreColorSystem/index.css';
[data-theme-mode="TORCH"] {
  --border-presentation-default-action-disabled: var(--black-500);
  --background-system-default-body-primary: var(--medium-red-600);
  --content-presentation-default-action-dark-primary: var(--purple-1000);
  --content-presentation-default-action-hover: var(--medium-red-1000);
  /* Additional vars... */
}
```

## Mods File

This file contains the different theme mods.

The component theme modes are:
    - dark mod
    - default mod
    - light mod
This mapping will be applied based on the HTML tag data-theme attribute.

#### Example of Mods File:

```css
/* Default theme */
[data-theme="default"] {
  --border-presentation-action-disabled: var(--border-presentation-default-action-disabled);
  --background-system-body-primary: var(--background-system-default-body-primary);
  --content-presentation-action-dark-primary: var(--content-presentation-default-action-dark-primary);
  --content-presentation-action-hover: var(--content-presentation-default-action-hover);
  /* Additional vars... */
}
```
