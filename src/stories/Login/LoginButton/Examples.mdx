  The LoginButton component is a customizable button that can be used for login actions. It supports all standard HTML button attributes and can be styled using the `className` prop.

## LoginButton Styles

### Default Style
<LoginButton style={{width:"300px"}}>Login</LoginButton>
<Source code={`<LoginButton>Login</LoginButton>`} />

## Usage

```jsx
import { LoginButton } from 'torch-glare';

const App = () => (
  <div>
    <LoginButton>Login</LoginButton>
    <LoginButton className="custom-class">Login</LoginButton>
    <LoginButton disabled>Login</LoginButton>
  </div>
);

export default App;
```
