The `Dynamic React Container` Component to contain HTML elements and change ti's direction when hit the viewport.
 this component use `useDirectionCalc` hook to change it's dirction when hit the viewport


## Used With DropDown

```jsx
import { DynamicContainer, DropDownMenu,DropDownMenuItem} from 'torch-glare';

const App = () => (
 <DynamicContainer active >
    <DropDownMenu component_style="System-Style">
      <DropDownMenuItem component_style='System-Style' component_label={'Option Example'} element_name={'example'} />
      <DropDownMenuItem component_style='System-Style' component_label={'Option Example 2'} element_name={'example2'} />
    </DropDownMenu>
</DynamicContainer>
);

export default App;
```
