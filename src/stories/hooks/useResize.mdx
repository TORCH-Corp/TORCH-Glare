# useResize

This hook is used to handle the resizing of a component. It manages mouse and touch events to calculate the new width based on the x coordinates. The hook returns the new width value and a function to start the resize process.


This hook is used with the `@dnd-kit/core` library to handle drag and drop events. It manages the `dragEnd` event and updates the data indexes accordingly. The data should be an array.
## Usege Example
<Source
  code={`
const App = () => {
      const resizableRef = useRef(null);
      const { width, handleStartResize } = useResize(resizableRef);

      return (
        <div>
          <div
            ref={resizableRef}
            style={{
              width: width || '200px',
              height: '100px',
              border: '1px solid black',
              position: 'relative',
              display: 'inline-block'
            }}
          >
            Resizable Box
            <div
              style={{
                width: '10px',
                height: '100%',
                background: 'gray',
                position: 'absolute',
                right: 0,
                top: 0,
                cursor: 'ew-resize'
              }}
              onMouseDown={handleStartResize}
              onTouchStart={handleStartResize}
            />
          </div>
          <div>Current Width: {width}px</div>
        </div>
      );
    }}
  `}
/>
