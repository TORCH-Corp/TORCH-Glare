This `RingLoading` component provides a visual indication of loading processes, featuring a rotating ring animation. It supports different sizes and allows for the inclusion of custom content within the animation frame.


## Examples

### Small Size
<RingLoading component_size="S">
  <span>Loading...</span>
</RingLoading>
<Source code={`<RingLoading component_size="S">
  <span>Loading...</span>
</RingLoading>`} />

### Medium Size

  <span>Please wait...</span>

### Large Size
<RingLoading component_size="L">
  <span>Processing your request...</span>

## Usage

```jsx
import {RingLoading }from 'torch-glare';

const App = () => (
  <div>
    <RingLoading component_size="M">
      <div>Loading data, please stand by...</div>
    </RingLoading>
  </div>
);

export default App;
```
