This `RadioLabel` component combines custom radio functionality with label management, providing a unified component for scenarios requiring radio buttons with extensive label options such as required and secondary labels.

## Examples

### Basic Usage
<RadioLabel
    component_size="M"
    name="basicRadio"
    label="Basic Radio"
/>

## Usage

```jsx
import { RadioLabel } from 'torch-glare';

const App = () => (
  <div>
    <RadioLabel
        component_size="M"
        name="exampleRadio"
        label="Example Radio"
        required_label="*"
        secondary_label="Optional"
    />
  </div>
);

export default App;
```
