### Dynamic Direction
Assuming the tooltip needs to adjust based on viewport constraints, the `useDirectionCalc` hook manages this dynamically, applying appropriate CSS classes to position.

## Usage in Application

```jsx
import { Tooltip } from 'torch-glare';

const App = () => (
  <div>
    <Tooltip message="Tooltip message" />
    <Tooltip message={null} /> {/* This will not render */}
  </div>
);

export default App;
```
