<Title>TableHeaderCell Component</Title>

The `TableHeaderCell` component represents individual header cells in a table. It supports resizing, sorting, and drag-and-drop functionality.

## Component Styles & Features

### Default Style
    <TableHeaderCell
        label="Column 1"
        component_size="M"
        id="col1"
      />

<Source
  code={`
    <TableHeaderCell
        label="Column 1"
        component_size="M"
        id="col1"
      />
  `}
/>


### Resize Feature
you can have a resizeable header by passing a resizable prop.
   <TableHeaderCell
        label="Column 1"
        component_size="M"
        resizable={true}
        id="col1"
      />

      <Source
  code={`
   <TableHeaderCell
        label="Column 1"
        component_size="M"
        resizable={true}
        id="col1"
      />
  `}
/>

### Sorting Feature 
You can use a sorting feature style by passing sorting_icon_type props and on_sorting_btn_click props.

  <TableHeaderCell
        label="Column 1"
        component_size="M"
        id="col1"
        sorting_icon_type="Default"
        on_sorting_btn_click={()=> alert("sorting function")}
      />

 <Source
  code={`
   <TableHeaderCell
        label="Column 1"
        component_size="M"
        id="col1"
        sorting_icon_type="Default" {/* Default, UP, DOWN, arrow directions */}
        on_sorting_btn_click={()=> alert("sorting function")}
      />
  `}
  />


## Example Usage

<Source

import { TableHeaderCell } from 'torch-glare';

          />

      </thead>
    </table>
  );
};

export default ExampleTableHeaderCell;
  `}
/>
