<Title>TableCell Component</Title>

The `TableCell` component represents individual cells in a table. It supports different sizes and can include custom children.
you can use it with label props to present the label only or put children's components.

## Component Styles & Features

### Default Style 

<TableCell cellLabel="Cell 1" component_size="M" />
<Source
  code={`
    <TableCell cellLabel="Row 1, Cell 1" component_size="M" />
  `}
/>


### With Child

<TableCell component_size="M" >
   <InputField name=''/>
</TableCell>

<Source
  code={`
<TableCell component_size="M" >
   <InputField name=''/>
</TableCell>
  `}
/>
