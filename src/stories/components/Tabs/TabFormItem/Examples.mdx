This `TabFormItem` component dynamically toggles between rendering as a `TopFormBarItem` or `SideFormBarItem` based on the `componentType` prop. This flexibility allows it to be utilized in various layout orientations, adhering closely to specified design guidelines.

## Examples

### Top Form Bar Item
<TabFormItem componentType="Top" >Label</TabFormItem>
<Source code={`<TabFormItem componentType="Top" >Label</TabFormItem>`} />

### Side Form Bar Item
<TabFormItem style={{width:"300px"}} componentType="Side">Label</TabFormItem>
<Source code={`<TabFormItem componentType="Side">Label</TabFormItem>`} />

## Usage

```jsx
import { TabFormItem } from 'your-component-library';

const App = () => (
  <div>
    {/* Use the TabFormItem as a top form bar item */}
    <TabFormItem componentType="Top" />

    {/* Use the TabFormItem as a side form bar item */}
    <TabFormItem componentType="Side" />
  </div>
);

export default App;
```
